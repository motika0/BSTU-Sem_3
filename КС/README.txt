# INSTRUCTIONS
1.  Import the OVA/VHDx into:
    VMware Player (PC & Linux users)
    VMware Fusion (Mac users)
    Hyper-V (Windows users)
    Alternatively VirtualBox, not recommended though. (PC, Linux, Mac)
2.  Login and become root in Ubuntu Server.

    [sudo] password for ncadmin: nextcloud

3. Run the script to do the final setup.
4. Enjoy your Nextcloud installation!
